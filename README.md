# Blackjack Game
A simple implementation of the classic Blackjack card game, designed for educational purposes. This project demonstrates object-oriented programming principles and game logic.

## 🛠 Configuration & Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/bohdan-bychak-sztu/blackjack
   ```
2. Navigate to the project directory:
   ```bash
   cd blackjack
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## 🚀 Basic Commands

### Run the Game
To start the application:
```bash
npm start
```

### Run Tests
To execute the test suite:
```bash
npm test
```
