# Privacy Policy

**Effective Date:** May 20, 2026  
**Project:** Blackjack (Web Application)  
**Author:** Bohdan Bychak

## 1. Introduction
This Privacy Policy explains how our project complies with the General Data Protection Regulation (GDPR). As a university project, we value your privacy and aim to be transparent about any data processed.

## 2. Data We Collect
This application is a client-side project. We **do not** collect or store any personal data on our servers.
* **Local Storage:** We use browser Local Storage to save your game progress (e.g., balance, wins) and UI preferences (e.g., language via `i18next`).
* **Cookies:** We use essential cookies to ensure the application functions correctly.

## 3. Purpose of Processing
The data processed is used exclusively for:
* Providing the core gameplay experience.
* Saving user settings between sessions.

## 4. User Rights (GDPR)
Under GDPR, you have the right to:
* **Access/Delete:** You can clear your data at any time by clearing your browser's cache and local storage.
* **Withdraw Consent:** You can manage your cookie preferences via the in-app popup.

## 5. Contact
If you have any questions regarding this project, please contact the author via the GitHub repository.
